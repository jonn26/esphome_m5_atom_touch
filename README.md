# esphome_m5_atom_touch
Esphome M5 Atom Touch Example

1. Install esphome
2. Clone or download these files into a directory
3. Edit secrets.yaml file with your wifi credentials and create an esphome rest & ota password (you can just create these and put them in the file, they will be used later to connect to home assistant and flash new firmware OTA)
4. flash your m5 atam esp32 with the command:
```
esphome run m5_atom_touch_headboard.yaml
```
